Шаг 2: Добавьте подписку в Happ / Hiddify

Универсальная подписка (VLESS + HY2):

https://redpillcloud.ru/sub/ВАШ_TELEGRAM_ID

Только HY2 (чистые Hysteria2-клиенты):

В Happ:

  1. Откройте Happ, перейдите в раздел «Подписки».
  2. Нажмите «+» → «Добавить по ссылке».
  3. Вставьте Subscription URL из бота, сохраните.

В Hiddify:

  1. Откройте Hiddify, нажмите «Добавить профиль».
  2. Выберите «Импорт из URL».
  3. Вставьте Subscription URL, сохраните, обновите профиль.

Важно: Subscription URL персональный и привязан к сроку подписки. Не делитесь им.

Подписка V2Box уже содержит все серверы (VLESS, HY2, RU/EU/NL) с правильными портами.

Шаг 2b: AWG — AmneziaWireGuard

AWG Windows macOS Linux Android

AmneziaWireGuard (AWG) — это модифицированный WireGuard с обфускацией (Junk-пакеты, смещение MTU, рандомизация handshake). Помогает обходить DPI там, где обычный WireGuard блокируют.

RedPill Cloud выдаёт AWG-конфиги на двух серверах:

  • 🇷🇺 RU — порт 8443 (Россия);
  • 🇪🇺 EU/NL — порт 8444 (Нидерланды).

Как получить AWG-конфиг

  1. Откройте @redpillcloud_bot.
  2. Нажмите «💳 Моя подписка» → «🔐 AWG (AmneziaWG)».
  3. Бот покажет кнопки: «📄 Скачать RU и 📄 Скачать EU».
  4. Скачайте конфиг-файл или скопируйте его содержимое.

Вариант A: Через Hiddify (рекомендуется)

  1. Скачайте и откройте Hiddify.
  2. Нажмите «Добавить профиль».
  3. Выберите «Импорт из буфера» или «Импорт из файла».
  4. Вставьте скопированный AWG-конфиг / выберите скачанный файл.
  5. Сохраните профиль. Hiddify сам распознает AWG и подставит нужные параметры.

Вариант B: AmneziaVPN (родной клиент)

  1. Установите AmneziaVPN с официального сайта.
  2. Запустите приложение, нажмите «Добавить сервер».
  3. Выберите «Импорт конфигурации».
  4. Выберите скачанный .conf-файл или вставьте текст конфига вручную.
  5. Подключитесь.

Вариант C: WireGuard (если нет DPI-блокировок)

AWG-конфиг совместим с обычным WireGuard. Если у провайдера нет глубокой фильтрации — просто импортируйте файл в WireGuard client.

Для Windows: если Hiddify показывает ошибку при импорте AWG-конфига — обновите Hiddify до последней версии или используйте AmneziaVPN.

Важно: AWG использует собственные Junk-параметры (Jc, Jmin, Jmax). Не меняйте их вручную.

Шаг 2c: TrustTunnel — TG Proxy / универсальный туннель

TrustTunnel Telegram Windows macOS Linux

TrustTunnel — это легковесный защищённый туннель поверх TLS (HTTPS). Идеален для обхода блокировок Telegram и другого трафика в сетях с жёстким DPI.

Сервер: tt.redpillcloud.ru:10444

Как получить доступ

  1. Откройте @redpillcloud_bot.
  2. Нажмите «💳 Моя подписка» → «🛰 TrustTunnel».
  3. Бот выдаст логин (формата rp_123456789) и пароль.

Настройка

Скачайте TrustTunnel-клиент:

TrustTunnel Client (GitHub)
  1. Скачайте TrustTunnel Client под вашу ОС с GitHub.
  2. Запустите клиент.
  3. Укажите сервер: tt.redpillcloud.ru:10444
  4. Введите логин и пароль из бота.
  5. Нажмите «Connect».

Быстрая настройка TrustTunnel вручную

Создайте файл client.toml:

server_address = "tt.redpillcloud.ru:10444"

[[client]]
username = "rp_ВАШ_ID"
password = "ВАШ_ПАРОЛЬ_ИЗ_БОТА"

Затем запустите:

trusttunnel_client client.toml

TrustTunnel работает поверх стандартного HTTPS (порт 10444), поэтому его сложно заблокировать — трафик неотличим от обычного веб-сёрфинга.

Шаг 3: Выбор сервера и правильные порты Hysteria2

После обновления подписки в клиенте появятся профили:

  • 🇳🇱 NL • RedpillCloud VLESS VLESS+REALITY
  • 🇷🇺 RU • RedpillCloud VLESS VLESS+REALITY
  • 🇳🇱 NL • RedpillCloud HY2 Hysteria2
  • 🇷🇺 RU • RedpillCloud HY2 Hysteria2
  • 🇷🇺 RU Mobile (экспериментальный) VLESS

Рекомендуем:

  • Для максимальной скорости и обхода блокировок — профили Hysteria2 (HY2).
  • Для стабильного доступа к российским сервисам — VLESS RU.

Порты для Hysteria2 (проверить в Happ/Hiddify, если не подставились автоматически):

  • 🇳🇱 NL HY2: порт 2443
  • 🇷🇺 RU HY2: порт 2444

Если после импорта клиент подставил другой порт — зайдите в настройки профиля и исправьте.

  1. Выберите профиль (например, «🇳🇱 NL • RedpillCloud HY2»).
  2. Проверьте порт: 2443 для NL, 2444 для RU.
  3. Нажмите «Подключиться».

Шаг 4: MTProto / MTProxy для Telegram

MTProto-прокси выдаются активным подписчикам. Используются два прокси — EU и RU.

  1. Откройте бота @redpillcloud_bot.
  2. Нажмите «💳 Моя подписка» → «🛰 MTProxy (2 прокси)».
  3. Выберите:
    • 📡 Прокси 1 (EU 8443) — сервер eu1.redpillcloud.ru;
    • 📡 Прокси 2 (RU 8080) — сервер redpillcloud.ru.
  4. Нажмите на кнопку — Telegram сам предложит подключить прокси.

Вручную: Настройки → Данные и память → Прокси → Добавить прокси → MTProto. Все параметры — из сообщения бота.

Шаг 5: SOCKS5-прокси (Smart TV, устройства без VPN-клиента)

SOCKS5 Smart TV Apple TV Консоли Любые устройства

Если на устройстве (Smart TV, приставка, игровая консоль) нельзя установить VPN-клиент — используйте SOCKS5-прокси. Вам понадобятся логин/пароль из бота.

  1. Откройте @redpillcloud_bot.
  2. Нажмите «💳 Моя подписка» → «🧦 SOCKS5».
  3. Бот покажет параметры:
    • Сервер: redpillcloud.ru
    • Порт: 1080
    • Логин и пароль (индивидуальные).
  4. Введите эти данные в настройках прокси вашего устройства.

Подсказка: На Android TV / Apple TV SOCKS5 настраивается в Wi-Fi-настройках сети → Прокси → Вручную. На телевизорах LG/Samsung — через системные настройки сети.

Альтернативные клиенты

Если Happ / Hiddify не подходят:

  • Android: v2rayNG, Nekobox, Nekoray (импорт через подписку по URL).
  • iOS: Shadowrocket, Stash (поддержка VLESS, импорт подписки).
  • Windows: v2rayN, Nekoray, Clash-клиенты (Clash Meta, cfw).
  • macOS / Linux: Clash-клиенты (Clash Verge, Clash Meta).

Для чистых HY2-клиентов бот даёт отдельную ссылку «Hysteria2 только».

Для AWG — AmneziaVPN под любую платформу.

Главное правило: используйте ваш Subscription URL — так вы всегда получите актуальный список серверов и обновления.

❓ Если что‑то не работает

Общие проблемы

  • Ссылки подписки скопированы полностью? Без пробелов и переносов?
  • Отключены другие VPN/прокси на устройстве?
  • Попробуйте другой профиль (другой город, другой протокол).
  • Перезапустите приложение и устройство.
  • Попробуйте мобильную сеть / другой Wi-Fi.

AWG не подключается на Windows

  • Обновите Hiddify до последней версии.
  • Попробуйте AmneziaVPN вместо Hiddify — он лучше работает с AWG.
  • Проверьте, что в конфиге указаны параметры Jc, Jmin, Jmax.
  • Убедитесь, что порты 8443 (RU) / 8444 (EU) не заблокированы провайдером.

TrustTunnel не соединяется

  • Проверьте логин и пароль — они чувствительны к регистру.
  • Убедитесь, что в клиенте указан правильный порт: 10444.
  • Проверьте, что адрес сервера: tt.redpillcloud.ru (без http://).
  • TrustTunnel использует TLS (порт 10444), обычный HTTPS-трафик не блокируется.

Hysteria2 не работает

  • Проверьте порт в настройках профиля: NL — 2443, RU — 2444.
  • Убедитесь, что выбран протокол Hysteria2 (не VLESS).
  • Для Happ/Hiddify: после импорта подписки обновите список серверов кнопкой Refresh.

Поддержка: если проблема не решилась — напишите в Telegram: @redpillcloud_bot.

К сообщению приложите: скрин настроек (сервер и порт), скрин ошибки, устройство, клиент, провайдера.
